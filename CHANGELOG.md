# Changelog

All notable changes to this project are documented here.

## [0.4.0] - 2026-07-17

### Added

- Read-only JSON workflow validation.
- Explicit command and HTTP authorization gates.
- Structured CLI error output.
- Distribution and clean-install checks.

## [0.3.0] - 2026-07-17

### Added

- JSON workflow specifications.
- File, HTTP JSON, and command workflow steps.

## [0.2.0] - 2026-07-17

### Added

- File artifact, HTTP JSON, and command adapters.
- Durable adapter receipts and uncertain-command reconciliation boundary.

## [0.1.0] - 2026-07-17

### Added

- SQLite workflow state.
- Stable step idempotency keys.
- Crash recovery and HMAC hash-chain audit verification.
